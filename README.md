**Collixa** is a state-of-the-art intent-based marketplace designed to bridge the gap between talent and opportunity. Leveraging AI-driven matchmaking and a robust credit-based "Wealth Protocol," Collixa empowers users to turn intentions into reality through seamless collaboration, secure transactions, and a community-driven "Tribe" architecture.

---

## ✨ Key Features

### 🧠 AI-Driven Matchmaking
Utilizes advanced generative AI (Google Gemini) to analyze user skills and intents, providing hyper-relevant suggestions for collaborators and projects.

### 💳 Wealth Protocol & Credit System
A sophisticated financial ecosystem featuring:
- **Credit Wallet**: Seamlessly manage platform credits for service exchange.
- **Voucher Vault**: A centralized hub to redeem, manage, and track rewards and marketplace vouchers.
- **Secure Payments**: Integrated with **Stripe** for reliable top-ups and withdrawals.

### 🛡️ Admin Control Center
A powerful dashboard for platform governance:
- **User Management**: Monitor user activity and handle reports.
- **Credit Oversight**: Granular control over credit distributions and transaction logs.
- **System Health**: Real-time insights into platform performance.

### 👥 Tribe Architecture
Move beyond simple profiles. Tribes allow users to form high-performance squads based on shared goals and complementary skill sets.

### 💬 Real-Time Collaboration
- **Integrated Chat**: Real-time messaging with presence indicators and multimedia sharing (Images, Location).
- **Legal Center**: Transparent and robust Privacy Policy and Terms of Service, ensuring a safe collaboration environment.

## 🚀 Getting Started

### Prerequisites
- **Node.js**: `v18.0.0` or higher
- **PostgreSQL**: Supabase project credentials
- **Environment Variables**: See `.env.example` in both folders.

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/mdarshadhussain/Collixa.git
   cd Collixa
   ```

2. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup**
   ```bash
   cd backend
   npm install
   # Configure your .env file
   npm run dev
   ```
